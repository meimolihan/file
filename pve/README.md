## CDN 加速地址

```markdown
wget -q -O /root/pve_source.tar.gz 'https://cdn.jsdelivr.net/gh/meimolihan/file@v1.0.0/pve/pve_source.tar.gz' && \
tar zxvf /root/pve_source.tar.gz && \
/root/./pve_source
```



- jsdelivr-CDN加速地址

- `https://cdn.jsdelivr.net/gh/`  ## 免费CDN加速GitHub  
- `meimolihan`  ## 用户名  
- `file`  ## 项目名  
- `img` ## 文件名  
- `v1.0.0`   ## 标签名  
- `git-01.jpg`  ## 文件名
